## 🌍 About

**BizPilot AI** is an AI-powered Business Intelligence and Management Platform built specifically for African Small and Medium Enterprises (SMEs).

Instead of juggling notebooks, spreadsheets, WhatsApp messages, and multiple software solutions, BizPilot AI brings everything together into one intelligent platform that helps business owners make faster, smarter, and data-driven decisions.

Our goal is simple:

> **Make world-class business intelligence affordable for every African entrepreneur.**

## 🎯 Why BizPilot AI?

African SMEs face several challenges:

- Limited access to business consultants
- Poor financial tracking
- Manual inventory management
- Guesswork in decision making
- Lack of business intelligence tools

BizPilot AI solves these problems by combining modern business management with Artificial Intelligence.

# 🚀 Quick Start

## Clone the repository

```bash
git clone https://github.com/yourusername/bizpilot-ai.git

cd bizpilot-ai
```

---

## Install dependencies

```bash
npm install
```

---

## Configure environment

```bash
cp .env.example .env
```

Update your environment variables.

---

## Start Development

```bash
npm run dev
```
